<b> Bold text – stylistically different without extra importance. This is bold text.
<strong> Strong importance – typically displayed as bold with semantic weight. This is important text.
<i> Italic text – alternate voice or mood (technical terms, thoughts, etc.). This is italic text.
<em> Emphasized text – adds stress emphasis with semantic meaning. This is emphasized text.
<mark> Highlighted text – marks text for reference, like a highlighter. This is highlighted text.
<small> Smaller text – for fine print or side comments. This is small text.
<del> Deleted text – strikethrough to show removal. This is deleted text.
<ins> Inserted text – underlined to show addition. This is inserted text.
<sub> Subscript text – lower and smaller, like in chemical formulas. H2O is water.
<sup> Superscript text – higher and smaller, like in exponents. E = mc2
<code> Inline code fragment – monospace font for code snippets. Use console.log() to debug.
<pre> Preformatted text – preserves spaces and line breaks exactly as written.

<abbr> Abbreviation – with optional full form in title attribute. HTML is awesome!
<cite> Citation – title of a referenced work like a book or article. From The Great Gatsby by F. Scott Fitzgerald.
<q> Short inline quotation with automatic quotation marks. He said, Hello world!
<blockquote> Block quotation – for longer quotes, typically indented.
To be or not to be.
